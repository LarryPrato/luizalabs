 * Introdução
 Seviço web (app) criado para visaulização de grafos, desenvolvido em python com a framework Flask. 
 Para o analisis  do grafo foi usada a biblioteca NetworkX. 
 A app  vem precarregada com uma grafo inicial, no entanto, novos vertices (nomes de pessoas) e novas 
 conexões poderão ser cadastrados, criando, em consequência, modificações a forma do grafo.

 * Requerimentos
    - Python 3
    - Flask
    - NetworkX
    - Pandas
    - Matplotlib
